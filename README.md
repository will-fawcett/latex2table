# latex2table
Create a pdf of a table from a latex table fragment.

# Build

## Clone
```
git clone https://github.com/will-fawcett/latex2table
```
Then: link the `latex2table.sh` script so that it is in your path. 

## Requirements
- pdflatex
- pdfcrop (usually comes with pdflatex)
